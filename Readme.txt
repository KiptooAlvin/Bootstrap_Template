# WANFAM Smart Livestock Management System

![WANFAM Logo](assets/img/favicon.png)

## Project Overview

WANFAM is a **Smart Livestock & Farm Management System** designed to help farmers and farm managers monitor, track, and improve farm productivity. The platform provides real-time insights, automated health alerts, feeding schedules, and resource optimization tools.

This project is a **portfolio details page** that showcases the WANFAM system's features and information using a modern, responsive web design built with **Bootstrap 5**.

## Features

- Responsive portfolio details layout for projects.
- Image slider showcasing dashboards, livestock tracking, health monitoring, and feeding management.
- Project information panel with category, client, date, and project URL.
- Fully responsive and mobile-friendly design.
- Integrated newsletter subscription form.
- Smooth animations with **AOS** and slider functionality with **Swiper.js**.

## Built With

- **HTML5 & CSS3** – Structure and styling
- **Bootstrap 5** – Responsive design framework
- **Swiper.js** – Portfolio image slider
- **AOS (Animate On Scroll)** – Scroll animations
- **JavaScript** – Interactive components
